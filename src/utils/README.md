### 工具类目录位置规范
·统一按照被作用的对象类型命名放置工具（就是按照名词）
例子：作用于鼠标事件的工具，应放置于event->mouseEvent目录下
如果同时存在多个作用对象比如鼠标事件和dom，按照返回值对象类型优先
